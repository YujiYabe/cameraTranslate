# cameraTranslate

## 概要
スマホカメラで写真→テキスト抽出→多言語翻訳するツール


## 機能一覧
- テキスト抽出
- 以下の言語に翻訳
    - 日本語
    - 英語
    - スペイン語
    - フランス語
    - ロシア語
    - 中国語-简体
    - 韓国語
    - アラビア語

## 使用技術
- google app engine 
- go 1.11
- vue.js
- google Cloud Vision API
- google Cloud Translation API
