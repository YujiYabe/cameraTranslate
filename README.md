# cameraTranslate

## 概要
スマホカメラで写真→テキスト抽出→多言語翻訳するツール


## 機能一覧
- テキスト抽出
- 多言語に翻訳
- 言語音声再生


## 使用技術
- google app engine
- go 1.11
- vue.js
- google Cloud Vision API
- google Cloud Translation API
